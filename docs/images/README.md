# OmniTAK GUI Screenshots

This directory contains screenshots of the OmniTAK graphical user interface.

## Screenshot Files

Please save your GUI screenshots with the following naming convention:

1. **gui-dashboard.png** - Dashboard view showing system metrics and connection status
2. **gui-connections.png** - Connections management page showing server list
3. **gui-messages.png** - Message viewer displaying CoT messages
4. **gui-settings.png** - Settings and configuration page

## Instructions for Adding Screenshots

To add screenshots to this directory:

1. Take screenshots of each GUI tab
2. Save them with descriptive names (e.g., `gui-dashboard.png`)
3. Place them in this `docs/images/` directory
4. Update the main documentation to reference these images

## Usage in Documentation

Reference these images in markdown files using:

```markdown
![Dashboard](images/gui-dashboard.png)
![Connections](images/gui-connections.png)
![Messages](images/gui-messages.png)
![Settings](images/gui-settings.png)
```

## Screenshot Guidelines

- Use PNG format for best quality
- Capture at a reasonable resolution (1280x720 or higher)
- Show representative data when possible
- Ensure no sensitive information is visible
- Use consistent window size across screenshots
